## Architecture

- **Content is the site**: Starlight maps `.md`/`.mdx` files in `src/content/docs/` to routes by file path (e.g. `guides/example.md` â†’ `/guides/example/`). The content collection schema is defined in `src/content.config.ts`.
- **Navigation**: the sidebar is configured in `astro.config.mjs`. The "Guides" section lists entries manually (new guide pages must be added there); the "Reference" section is autogenerated from the `reference/` directory.
- **Assets**: images referenced from Markdown go in `src/assets/`; static files (favicons, etc.) go in `public/`.
